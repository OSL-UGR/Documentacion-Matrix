Para la capa visual de los usuarios, se descartaron directamente otras alternativas a favor de **Element**.

Element es el cliente más utilizado del ecosistema Matrix. La decisión de adoptarlo es parecida a la que ya se tomó con Synapse. Ofrece un **buen soporte y una buena documentación**. Además incorpora de forma nativa herramientas para la gestión de salas, roles de usuarios, creación de comunidades etc

- **Código fuente:** [Repositorio oficial de Element](https://github.com/element-hq/element-web/tree/develop/apps/web)
