
En este caso de uso se describe el procedimiento para acceder directamente a la base de datos interna mediante la consola si se prefiriera realizar consultas SQL directamente sobre esta..

1. **Requisitos previos:**
	
	- Tener acceso SSH con permisos de sudo a la maquina.
	- Situarnos en el directorio `cd /home/svc`
	
2. **Procedimiento:**

- **a) Abrimos la consola SQL:**

Ejecutamos el siguiente comando para acceder a la consola de nuestra base de datos:

``` bash
sudo sqlite3 data/homeserver.db 
```

- **b) Ejecutamos las consultas necesarias:**

Una vez dentro el curso cambiará a `sqlite>` indicando que ya podemos introducir las sentencias SQL necesarias. Por ejemplo para mostrar rápidamente a los usuarios registrados en el sistema podemos ejecutar:

``` SQL
SELECT * FROM users;
```

c) **Salida segura:**

Para salir de forma segura y evitar algún bloqueo en la base de datos, en vez de ejecuttar `Ctrl+C` es recomendable ejecutar el siguiente comando en la consola de sqlite:

``` bash
.quit
```
