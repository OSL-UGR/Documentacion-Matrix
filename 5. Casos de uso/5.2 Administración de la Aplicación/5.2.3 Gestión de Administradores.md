
En este caso de uso se describe el procedimiento por el cual podemos otorgarle a un usuario del servidor privilegios de administrador, para que este también pueda acceder al panel d control y gestionar el sistema.

1. **Requisitos previos:**
	
	- Tener abierta la interfaz de Synapse-Admin con permisos de administración (si no se conoce el procedimiento está explicado en el caso de uso "5.2.1 Acceso a Synapse-Admin").
	
2. **Procedimiento:** 

- **a) Encontrar al usuario:**

En el menú lateral dos dirigimos a la sección donde se listan todos los usuarios del servidor. Podemos buscar entre todo ese listado o simplemente filtrar mediante el buscador introduciendo el ID del usuario que queramos encontrar. Una vez encontramos clicamos en el.

- **b) Le damos permisos de admistración:**

Dentro de la pestaña "USER" del usuario que nos muestra su información general, si nos desplazamos hacia abajo debemos encontrar la opción de "Server Administrator" que tenemos que activar para designar a ese usuario como administrador.

**Nota:** Aunque aquí se ha explicado cómo hacerlo a través de la interfaz gráfica, cabe recalcar que esta acción también se puede realizar a través de consultas SQL interactuando directamente con la base de datos desde la terminal.
