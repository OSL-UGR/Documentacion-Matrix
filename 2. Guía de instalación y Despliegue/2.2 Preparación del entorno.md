
Matrix se ha desplegado utilizando contenedores, por lo que el primer paso fue actualizar el sistema base e instalar Docker. Al tratarse de un sistema Red Hat es necesario el gestor de paquetes "dnf", a diferencia del apt al que muchos están acostumbrados.


**1. Actualización e instalación de dependencias base:**

```bash
sudo dnf update
sudo dnf install -y dnf-plugins-core curl gnupg
```

**2. Instalación de Docker y Docker Compose:** Añadimos el repositorio oficial de Docker e instalamos los paquetes necesarios.

```bash
sudo dnf config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
sudo dnf install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

**3. Activación del servicio:** Para permitir que Docker se ejecute automáticamente tras cualquier reinicio de la máquina, se habilitó y arrancó el servicio mediante "systemctl".

```bash
sudo systemctl start docker
sudo systemctl enable docker
sudo systemctl status docker
```
