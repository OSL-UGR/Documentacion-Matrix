


**Fase 1: Los Plugins de Maubot (Lo más rápido y directo)**

1. **`github` (Plugin Maubot):** Para que las notificaciones de los commits caigan limpiamente en la sala de desarrollo correspondiente, sin la complejidad de Hookshot.
    
2. **`rss` (Plugin Maubot):** Para conectar el blog/web oficial a una sala de avisos.
    
3. **`FAQbot` (Plugin Maubot):** (El que comentamos antes, muy útil para estudiantes).
    

**Fase 2: Bots Dedicados (Los pesos pesados)**

4. **`Honoroit`:** Para el sistema de soporte y tickets.

5. **`matrix-email-bot` o `gmailhooks` (Plugin):** Para convertir correos en mensajes de Matrix.

6. **`baibot`:** Para trastear con IA (con prioridad baja).

¡Pues manos a la obra con la **Fase 1**! Como ya tienes Maubot instalado, con el cifrado apagado, y el usuario `@bot.mauubot:matrix.ugr.es` funcionando a la perfección en el Chat General, añadir el plugin de GitHub y RSS es literalmente cuestión de un par de clics y un archivo de configuración.
