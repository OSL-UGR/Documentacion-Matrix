
Este archivo es un documento estático proporcionado directamente por el CSIRC, que es esencial para el funcionamiento de inicio de sesión único SSO.

Este archivo XML es necesario para que Matrix confíe en el inicio de sesión de la UGR, conteniendo las urls a las que se redireccionan para el inicio de sesión, la identidad del emisor (idP) y los certificados necesarios.

**MUY IMPORTANTE:** Sin este archivo en el directorio data/, o si el certificado de la universidad caducara y no se actualizara en este documento, el acceso mediante SSO quedaría inhabilitado.
