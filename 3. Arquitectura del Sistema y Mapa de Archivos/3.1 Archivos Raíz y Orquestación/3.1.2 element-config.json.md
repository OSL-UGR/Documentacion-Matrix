
Este archivo es el encargado de personalizar la interfaz web que ven los usuarios. Permite "capar" opciones que no nos interesan dentro de nuestra configuración además de darle un aspecto visual relacionado con la universidad.

Este archivo tiene la siguiente forma:


```json
{
    "default_server_config": {
        "m.homeserver": {
            "base_url": "https://matrix.ugr.es",
            "server_name": "matrix.ugr.es"
        }
    },
    "brand": "Matrix UGR",
    "branding": {
        "auth_header_logo_url": "https://ecalfor.eu/wp-content/uploads/2021/06/logo-ugr-red-black.png",
        "welcome_background_url": "https://upload.wikimedia.org/wikipedia/commons/c/c1/Hospital_Real_de_Granada_-_Espa%C3%B1a.jpg"
    },
    "disable_custom_urls": true,
    "disable_guests": true,
    "disable_3pid_login": true,
    "disable_login_language_selector": true,
    "default_theme": "light",
    "help_url": "https://osl.ugr.es/",
    "terms_and_conditions_links": [
        {
            "text": "Política de privacidad",
            "url": "https://www.ugr.es/~edafolo/privacidad.php"
        }
    ],
    "sso_redirect_options": {
        "immediate": false,
        "on_login_page": true
    },
    "setting_defaults": {
        "UIFeature.registration": false,
        "UIFeature.passwordReset": false,
        "UIFeature.deactivate": false,
        "UIFeature.thirdPartyId": false,
        "UIFeature.allowCreatingPublicRooms": true,
        "UIFeature.allowCreatingPublicSpaces": true,
        "custom_themes": [
            {
                "name": "UGR Corporativo",
                "is_dark": false,
                "colors": {
                    "accent-color": "#CB2C30",
                    "primary-color": "#CB2C30"
                }
            }
        ]
    }
}
```

## Explicación de la Configuración

La configuración se divide en varios bloques para controlar el comportamiento y el aspecto visual de Element:

- **default_server_config**: Fija nuestro servidor (`matrix.ugr.es`) como el servidor base. Esto evita que el cliente intente conectarse al servidor público de `matrix.org` por defecto.
- **brand** y **branding**: Define el nombre de la pestaña ("Matrix UGR") y carga los recursos visuales externos para la página de inicio de sesión como el logo de la UGR y una imagen de fondo del Hospital Real.

- **Bloqueo de funciones (disable_...)**: Vamos a establecer varias opciones en true para restringir el uso de la aplicación.

    - **disable_custom_urls**: Impide que el usuario cambie la URL del servidor en la pantalla de inicio.
    - **disable_guests**: Deshabilita el acceso de cuentas invitadas.
    - **disable_3pid_login**: Oculta los campos tradicionales de inicio de sesión con correo/contraseña o teléfono, forzando el uso del SSO de la universidad.
    - **disable_login_language_selector**: Oculta el selector de idioma en la pantalla principal.
    
- **default_theme**: Establece el tema claro (`light`) por defecto al cargar la página.
- **help_url** y **terms_and_conditions_links**: Añade enlaces de interés en la interfaz, dirigiendo a la web de la OSL para soporte y a la política de privacidad de la universidad.
- **sso_redirect_options**:
    
    - `immediate: false` y `on_login_page: true` aseguran que el usuario vea la pantalla de inicio con el fondo y el logo, mostrando el botón de inicio de sesión único (SSO).
    
- **setting_defaults**: Este bloque oculta y desactiva elementos de la interfaz de usuario (`UIFeature`):
    
    - Se ocultan los botones de registro (`registration`), cambio de contraseña (`passwordReset`), vinculación de terceros (`thirdPartyId`) y desactivación de cuenta (`deactivate`).
        
    - Se permite a los usuarios crear salas públicas (`allowCreatingPublicRooms`) y espacios públicos (`allowCreatingPublicSpaces`).
        
    - **custom_themes**: Inyecta un tema estético adicional llamado "UGR Corporativo", configurando los botones y acentos de la aplicación con el código de color rojo oficial de la Universidad.
